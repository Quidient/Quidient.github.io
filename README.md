# Quidient Github Pages

## Structure
```
index.html                 
static/
  css/
    index.css              
    bulma.min.css          vendored libraries 
    bulma-carousel.min.css
    bulma-slider.min.css
    academicons.min.css
  js/
    index.js               carousel + BibTeX-copy logic
    jquery.min.js
    fontawesome.all.min.js
    bulma-carousel.min.js
    bulma-slider.min.js
  fonts/                   academicons webfonts
  images/                  placeholder SVGs 
  videos/                  drop result videos here
```

## Notes
- Icons use Font Awesome (`fas`/`fab`) and Academicons (`ai`). The footer credits
  both, as their licenses (CC BY 4.0 / SIL OFL 1.1) ask; the template itself needs none.
- Fonts, MathJax, and Google Fonts load from CDNs; everything else is local.
